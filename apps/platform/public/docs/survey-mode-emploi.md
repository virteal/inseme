# Mode d'emploi Kudocracy.Survey

Ce guide pratique vous accompagne pas à pas pour utiliser la plateforme citoyenne. Il complète la
[présentation](docs/survey.md) du projet et se concentre sur les actions concrètes à réaliser.

## 1. Préparer votre participation

1. **Créer un compte** : déroulez la barre qui s'ouvre en bas de page puis cliquez sur le bouton «
   Connexion / Inscription » pour lancer l'assistant d'authentification.
2. **Valider votre email** : cliquez sur le lien reçu (sauf si connexion via Facebook).
3. **Compléter votre profil** : sur [Votre profil](/profile), indiquez quartier, centres d'intérêt
   et liens utiles afin que les autres membres puissent vous identifier facilement.

## 2. Comprendre la navigation

- **Tableau de bord social** [Votre tableau de bord](/social-dashboard) : synthèse de vos articles,
  commentaires, statistiques et raccourcis d'action.
- **Café Pertitellu** [Café](/social) : fil social, groupes thématiques, discussions reliées aux
  éditions de la Gazette.
- **Gazette** [Gazette](/gazette) : articles éditorialisés avec accès direct aux groupes et
  propositions associées.
- **Wiki collaboratif** [Wiki](/wiki) : bases de connaissances partagées.
- **Kudocratie** [Proposition](/kudocracy) : dépôt et vote des propositions.
- **Consultations** [/consultations](/consultations) : questionnaires temps réel.

## 3. Scénarios clés

### Publier au Café ou dans la Gazette

1. Depuis [Vos publications](/social?tab=posts), cliquez sur **"Écrire un article"**.
2. Sélectionnez le groupe ou la Gazette cible dans le panneau latéral (optionnel mais recommandé).
3. Rédigez votre contenu, ajoutez des médias, précisez les tags.
4. Publiez ; l'article apparaîtra dans le Café et, si une Gazette est choisie, sur la page de
   l'édition correspondante.

### Rejoindre un groupe et suivre ses discussions

1. Dans [Groupes](/social?tab=groups), parcourez les groupes (quartier, association, thématique).
2. Cliquez sur **"Rejoindre"** pour devenir membre.
3. Une fois membre, les discussions du groupe sont prioritaires dans votre fil et accessibles via
   [Le Café](/social).

### Proposer ou soutenir une idée dans la Kudocratie

1. Ouvrez [Propositions](/kudocracy) et sélectionnez **"Nouvelle proposition"**.
2. Décrivez le problème, proposez une solution, rattachez le thème adéquat.
3. Publiez : les autres citoyens peuvent voter (pour/contre/abstention) ou déléguer leur voix.

### Répondre à une consultation

1. Allez sur [/consultations](/consultations) pour voir les questionnaires ouverts.
2. Cliquez sur la consultation voulue.
3. Répondez aux questions ; vos réponses sont anonymisées et agrégées.
4. Les graphiques s'actualisent en temps réel une fois votre participation enregistrée.

### Utiliser Ophélia, l'assistante IA

1. Ouvrez [IA](/assistant).
2. Posez votre question (ex. "Comment proposer une piste cyclable ?").
3. Ophélia vous renvoie des extraits du wiki, des propositions ou des discussions pertinentes.

## 4. Suivre vos contributions

- Consultez [Vos contribution](/social-dashboard) pour un récapitulatif personnalisé.

## 5. Bonnes pratiques

- Restez cordial et factuel ; la modération veille à préserver l'espace de dialogue.
- Citez vos sources dans les propositions et articles.
- Utilisez les tags/groupes pour que vos contenus soient retrouvés.
- Avant de créer une nouvelle proposition, vérifiez si un sujet similaire existe déjà.

## 6. Besoin d'aide ?

- **Assistance** : contactez [Contact](contact) ou utilisez le bot Ophélia.
- **Communauté** : rejoignez le groupe Facebook Pertitellu pour des échanges informels.
- **Tech** : ouvrez une issue GitHub dans le dépôt `JeanHuguesRobert/survey`.

Bon usage de Kudocracy.Survey ! Ensemble, faisons vivre la participation citoyenne.
