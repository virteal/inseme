Instructions pour la suppression des données (Facebook)

Page publique

Si vous souhaitez demander la suppression des données liées à votre compte Facebook, suivez ces étapes :

  1. Dans Facebook, utilisez l'outil de confidentialité pour demander la suppression des données de l'application.
  2. Facebook enverra alors une requête à notre serveur. Nous générerons un code de confirmation et mettrons à jour le statut de la demande.
  3. Pour vérifier le statut : ouvrez /oauth/facebook/deletion-status et ajoutez le paramètre ?code=VOTRE_CODE fourni par Facebook.

Exemple :
/oauth/facebook/deletion-status?code=012345abcdef

Si vous ne recevez pas de code ou avez besoin d'aide, contactez-nous.
